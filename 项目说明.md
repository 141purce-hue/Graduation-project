# 🌿 番茄病虫害智能识别系统 - 项目说明

## 📌 项目概述

这是一个基于 **YOLOv8** 深度学习模型和 **PyQt5** 图形界面的番茄病虫害智能识别系统。系统提供了直观美观的用户界面，支持图片和视频两种识别模式，能够准确识别四种常见的番茄叶片病害。

## 🎯 项目特点

### 1. 精美的用户界面
- ✨ 现代化设计风格，采用扁平化设计理念
- 🎨 科学的配色方案，视觉舒适
- 📐 合理的布局结构，左右分栏设计
- 🖱️ 流畅的交互体验，按钮状态清晰

### 2. 双模式识别
- 📷 **图片识别模式**：支持单张图片的病害检测
- 🎥 **视频识别模式**：支持视频流的实时病害检测

### 3. 四类病害识别
- 🟢 **Healthy Leaf**（健康叶片）
- 🟠 **Leaf Mold**（叶霉病）
- 🔴 **Septoria leaf spot**（斑点病）
- 🟣 **Tomato leaf bacterial spot**（细菌性斑点病）

### 4. 灵活的配置系统
- ⚙️ 独立的配置文件，方便参数调整
- 🔧 支持自定义模型路径、置信度阈值等
- 🎨 支持自定义界面颜色和窗口大小

### 5. 完善的文档体系
- 📖 详细的使用说明

- 🎨 界面设计文档
- 🔍 环境检查工具

## 🚀 核心功能

### 1. 图片识别
```python
# 工作流程
选择图片 → 加载图片 → YOLO 推理 → 绘制结果 → 显示结果
```

**特点**：
- 支持多种图片格式（JPG, PNG, BMP）
- 自动缩放适应显示区域
- 显示检测框和置信度
- 彩色文本区分不同病害

### 2. 视频识别
```python
# 工作流程
选择视频 → 逐帧读取 → YOLO 推理 → 绘制结果 → 实时显示
```

**特点**：
- 多线程处理，界面不卡顿
- 实时显示识别结果
- 支持暂停和停止
- 自动释放资源

### 3. 结果展示
- 📊 置信度百分比显示

## 📊 性能指标

### 识别速度
- **图片识别**：< 1 秒/张（取决于图片大小）
- **视频识别**：实时处理（30 FPS）

### 准确率
- 取决于训练模型的质量
- 建议置信度阈值：0.5

### 资源占用
- **内存**：约 500MB - 2GB（取决于模型大小）
- **CPU**：中等占用
- **GPU**：支持 CUDA 加速（可选）
